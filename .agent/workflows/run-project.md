# Running the Project

1. **Install dependencies**
```bash
npm install
```

2. **Start the development server**
// turbo
```bash
npm run dev
```
Open [http://localhost:4321](http://localhost:4321) in your browser.

3. **Build for production**
```bash
npm run build
```

4. **Preview the production build**
```bash
npm run preview
```
